User guide

How to create a new Scrum Team?

  1. In the Project app, go to Configuration/Scrum Teams.
  2. Click on "Create"
  3. A Wizard opens where you can fill the name of your new team as well as assign it to a sprint and a project.

How to create a new Sprint?

  1. In the project app go to Search/Sprints.
  2. Click on create.
  3. Fill in the wizard, all fields are mandatory except the check boxes.
  4. Once a Sprint is created you can click on it and see its details. Note that in the details you have a button giving a list of the linked tasks as well as a link to the Scrum Team details assignated to the Sprint.
